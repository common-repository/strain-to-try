== Description ==

This diplays a random medical marijuana strain to try in the footer.

== Installation ==

1. Upload `strain.php` to the `/wp-content/plugins/` directory
1. Upload `strain.txt` to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently Asked Questions ==

= How do I add strains to recommend =

Go to the location of the plugin and there is a file named strain.txt in there is a list a -z of strains add your strains in.
